# To-Do List

Веб-приложение для управления задачами.

## Возможности

- Добавление задач
- Удаление задач
- Редактирование задач
- Отметка выполнения
- Поиск задач
- Фильтрация задач
- Сохранение данных в LocalStorage

## Используемые технологии

- HTML5
- CSS3
- JavaScript
